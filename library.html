Library Management System

During my second year of college in the "IPT 1" course, I developed a Library management system comprising login, search, home, add book, borrow book, return book, book records, and data interfaces, allowing authorized users to efficiently manage library resources, search for materials, add new items, handle checkouts and returns, access detailed book information, and maintain the system's data.

Page 1

Login - This is where the admin can log in to their account.

Page 2

Home - This is the base interface, or the main dashboard.

Page 3

Add book - The admin can add books to the database based on the books that are in the library.

Page 4

Borrow Book - This is where the borrowing of books happens, and the admin will fill up the necessary information.

Page 5

Return book - This is where the admin can modify the data and set the book as returned.

Page 6

Book Records - Displays all books so that the admin can archive them.

Page 7

Data - Collection of borrowed and returned books.
